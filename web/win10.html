Windows10 단축키

Win + UP Arrow : 현재 창을 최대화. 
Win + Down Arrow : 현재 창을 최대화에서 복원 하거나 최소화.
Win + Home : 현재창의 제외한 창을 최소화.
Win + Space : 전체창을 투명화 해서 바탕화면을 봄.
Win + Pause/Break : 제어판의 시스템 열기. 
Win + D : 바탕화면 보기/감추기. 
Win + E : 윈도운 탐색기 실행.
Win + L :  컴퓨터 잠금.
Win + M : 모든 윈도우 최소화.
Win + Shift + M : 전체창 최소화 복원.(Win + M 반대) 
Win + (+) / (-) : 윈도우 확대/축소
Win + PrintScreen : 전체화면 캡쳐
Win + Alt + PrintScreen : 현재창 캡쳐
Win + Shift + s : 특정영역 캡쳐  
검색착에서 캡처도구를 치면 기본 캡처프로그램이 구동된다.
